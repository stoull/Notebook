# Swift - Combine framework


函数式响应式编程FRP (Functional Reactive Programming)是一种编程范式，它采用函数式编程的基础部件（如map、reduce、filter等），进行响应式编程（异步数据流程编程）。FRP被用于GUI、机器人和音乐方面的编程，旨在通过显式的建模时间来简化这些问题。

### Combine
