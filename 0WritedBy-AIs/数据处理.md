### 转成ChatLab支持的格式-ChatDataTransformer

在/Users/hut/Desktop/ChatDataTransformer目录下有三个文件，分别是'chat_redacted.json','chatlab-format.md'和'PrivateChatExample.json'。'chat_redacted.json'分文件中存储了很多导出的两个人的微信聊天信息，类似如下：
```
{
      "local_id": 22,
      "timestamp": 1477833293,
      "sender": "USER_PEER",
      "content": "车上的人看着我们一起玩都说不知道的还以为我们是母子[Chuckle]"
    },
    {
      "local_id": 23,
      "timestamp": 1477833700,
      "sender": "USER_ME",
      "content": "我想也是"
    },
```
不要全读'chat_redacted.json'中的所有内容，读前面的就行，消息类型有语音，图片，视频通话等。
