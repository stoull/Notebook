# 常用工具

### 图片相关

Linux 系统：`imagemagick `

安装：`brew install imagemagick`

查看：`mogrify -version` 或者 `mogrify`

#### 转换格式

##### 单个图片

`mogrify -format jpg MY-PNG-FILE.png`

##### 多个图片

`mogrify -format jpg *.png`
